# Linux Permissions Quiz (SUID/SGID/Sticky/umask/ACL)

Sitio web interactivo para practicar permisos en Linux. Corre 100% estático (HTML/CSS/JS).  
**Demo**: habilita GitHub Pages en `Settings → Pages` y selecciona `Branch: main` + `Folder: /docs`.

## Contenido
- 4 preguntas de máxima dificultad sobre permisos en Linux
- Corrección automática + explicaciones
- Responsive y accesible (sin frameworks)

## Cómo publicar en GitHub Pages
1. Sube este repo a GitHub.
2. Ve a **Settings → Pages**.
3. En **Source**, elige **Deploy from a branch**.
4. **Branch:** `main` – **Folder:** `/docs` → **Save**.
5. Espera unos segundos: aparecerá la URL pública.

## Desarrollo local
Solo abre `docs/index.html` en el navegador.
